# 🏨 Lagos Prestige | Premium Shortlet & Hospitality Engine

**Lagos Prestige** is a high-fidelity hospitality platform engineered for the luxury shortlet market in Nigeria. It combines a sophisticated "split-pane" browsing experience with real-time availability tracking and seamless booking workflows.

## 🚀 The Tech Stack
* **Framework:** React 19 + Vite 6 (Maximum Frontend Velocity)
* **State Management:** TanStack Query v5 for instant, cached property listings.
* **Booking Logic:** `date-fns` + `react-day-picker` for robust reservation scheduling.
* **UI/UX:** `react-resizable-panels` for desktop "Map + List" views and `vaul` for mobile-native drawers.
* **Animations:** Framer Motion 12 for a high-end "boutique" feel.

## 🧠 Key Features
* **Dynamic Availability Engine:** Utilizes TanStack Query to ensure users only see currently available luxury units, reducing booking friction.
* **Interactive Discovery:** Features a modular, resizable interface that allows travelers to compare properties and amenities side-by-side.
* **Mobile-First Hospitality:** Deeply optimized for Lagos-based users, featuring gesture-based navigation and lightweight asset loading.
* **Type-Safe Reservation Flow:** Leverages Zod for strict validation of guest data and booking intervals, ensuring data integrity across the platform.

## 🛠️ Performance Architecture
Designed to handle the high-traffic demands of peak travel seasons. By utilizing the Vite 6 build engine and React 19's concurrent rendering, Lagos Prestige delivers an ultra-responsive experience that matches the premium quality of the properties it showcases.
